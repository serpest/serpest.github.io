MusicSync

2020-2021

MusicSync is a cross-platform tool that synchronizes music libraries between devices and folders using filters, like rating, genre and release year.
It suppport two transfer protocols: MSC (Mass Storage Class) and ADB (Android Debug Bridge).

The GUI was realized using Qt for Python.

TicTacToePlus

2019-2020

TicTacToePlus is an highly customizable Tic-tac-toe game, where you can decide the number of players, the grid size and the tic-tac-toe number.

The main challenges of this program are finding a tic-tac-toe in the grid and developing a game strategy for the different difficulty levels.

The GUI was realized using Java AWT.
